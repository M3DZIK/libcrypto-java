A cryptography utilities for Java

## Installation

### Maven

```xml
<dependency>
    <groupId>dev.medzik</groupId>
    <artifactId>libcrypto</artifactId>
    <version>0.2.0</version>
</dependency>
```

### Gradle

### Groovy

```groovy
dependencies {
    implementation 'dev.medzik:libcrypto:0.2.0'
}
```

### Kotlin

```kotlin
dependencies {
    implementation("dev.medzik:libcrypto:0.2.0")
}
```
